# 第一天：递归实现指数型、全排列、组合数枚举
题目[递归实现指数型枚举](https://www.acwing.com/problem/content/94/)
## 思路
先看数据范围，显然数据范围较小，因此可以选择dfs算法——深度优先搜索
### 题目分析：
有n个数，要求对其可能的策略进行统计，即每种都是选或者不选，共有2^n种可能（也正好符合对时间复杂度的分析）现在好比有n个萝卜坑，统计有多少种拔萝卜的情况，现在每个坑进行一个统计，需要记录每个坑的状态，“未选”“被选”“不选” 如何统计状态？——利用一个数组 `st[N]` 用作状态数组 0代表未选 1代表被选 2代表不选。
写递归要注意边界的判断，以便于及时终止，同时dfs一般都是用全局变量——>更方便
```C++
#include <bits/stdc++.h>

const int N = 15;
int n;//实际的数据量
int st[N];//记录状态， 0表示未考虑，1表示选，2表示不选

void dfs(int u)//u为搜索的层数
{
  if(u > n)//边界情况
  {
        for(int i = 1; i <= n; i++)
        {
          if(st[i]==1) printf("%d ", i);
        }
            printf("\n");      
            
                return;
  }


  //模拟分支的选择与否


    st[u]=1;
    dfs(u+1);
    
    st[u]=0;//恢复现场

    st[u]=2;
    dfs(u+1);
    
    st[u]=0;//恢复现场

}

int main (){
    scanf("%d", &n);
    
    dfs(1);

    return 0;
}
```
